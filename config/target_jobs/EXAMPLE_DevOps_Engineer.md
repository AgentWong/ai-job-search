# DevOps Engineer - Acme Cloud Services

**Company:** Acme Cloud Services
**Location:** Remote (US)
**Salary:** $100,000 - $130,000

## About Acme Cloud Services

Acme Cloud Services is a mid-size SaaS company (350 employees) providing cloud-based project management tools for construction companies. Our platform processes over 2 million API requests daily and serves 50,000+ active users. We're a remote-first company with team members across 30 states.

## About the Role

We're looking for a DevOps Engineer to join our Infrastructure team (5 engineers). You'll be responsible for maintaining and improving our AWS infrastructure, building CI/CD pipelines, and automating operational processes. You'll work closely with our development teams to ensure reliable, scalable deployments.

## Responsibilities

- Manage and improve AWS infrastructure including EC2, ECS, RDS, S3, and CloudFront
- Write and maintain Terraform configurations for all infrastructure resources
- Build and optimize CI/CD pipelines using GitHub Actions
- Implement monitoring and alerting using Datadog and PagerDuty
- Manage Ansible playbooks for server configuration and patching
- Participate in on-call rotation (1 week per month)
- Automate operational tasks using Python and Bash scripts
- Collaborate with development teams on deployment strategies
- Maintain infrastructure documentation and runbooks

## Requirements

- 3-5 years of experience with AWS cloud infrastructure
- Strong experience with Terraform for infrastructure-as-code
- Experience with Ansible or similar configuration management tools
- Proficiency with CI/CD tools (GitHub Actions, Jenkins, or GitLab CI)
- Solid Linux administration skills (RHEL or Ubuntu)
- Scripting experience in Python and/or Bash
- Understanding of networking concepts (VPC, subnets, security groups, load balancers)
- Experience with containerization (Docker, ECS or Kubernetes)

## Nice to Have

- AWS Solutions Architect certification
- Experience with Datadog or similar observability platforms
- Experience with blue-green or canary deployment strategies
- Familiarity with cost optimization practices

## Benefits

- Fully remote work environment
- Health, dental, and vision insurance
- 401(k) with 4% match
- Unlimited PTO (minimum 3 weeks encouraged)
- $1,500 annual learning budget
- Home office stipend
